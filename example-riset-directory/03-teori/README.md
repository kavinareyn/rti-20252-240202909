## Ringkasan

Folder 03-teori menjadi dasar konseptual penelitian ini. Seluruh teori, desain penelitian, dan arsitektur sistem yang dijelaskan pada folder ini digunakan sebagai acuan dalam implementasi Convolutional Neural Network (CNN) untuk mengklasifikasikan 10 kelas penyakit daun tomat menggunakan Tomato Diseases Dataset dari Kaggle.

Pembahasan mencakup proses preprocessing citra, pembangunan dan pelatihan model CNN, evaluasi performa menggunakan Accuracy, Precision, Recall, F1-Score, Classification Report, dan Confusion Matrix, serta penerapan Gradient-weighted Class Activation Mapping (Grad-CAM) untuk memvisualisasikan area citra yang menjadi dasar pengambilan keputusan model. Dengan demikian, penelitian tidak hanya menghasilkan model klasifikasi yang akurat, tetapi juga memberikan interpretasi visual yang meningkatkan transparansi dan kemudahan dalam memahami hasil prediksi model.
